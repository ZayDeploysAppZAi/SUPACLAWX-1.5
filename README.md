# SUPACLAWX 1.5 — NemoClaw Intelligence Vault

An AI-powered content and course generation platform with store publishing, PDF generation, vlog creation, avatar management, and email delivery.

## Features

- **Store & Admin Pages** — Auto-generated HTML store and admin dashboards
- **Course Generator** — AI-powered course creation pipeline
- **PDF Generator** — Dynamic PDF creation and preview
- **Vlog Creator** — Automated vlog content generation
- **Avatar Manager** — AI avatar management system
- **Image Generation** — Bulk and hero image generation
- **Teaser Engine** — Promotional teaser content creation
- **Email Sender** — Automated email delivery system
- **Video Generation** — Hero video production pipeline
- **Storage** — Persistent data storage layer
- **MU API Client** — External API integration

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
# Build store and admin pages
python3 main.py

# Start the server
python3 server.py
```

## Stack

- Python 3
- httpx, Pillow, fpdf2, PyMuPDF, stripe, rich, schedule
